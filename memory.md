# Memory

This document stores accumulated insights, patterns, and lessons from your reviews and reflections. It serves as your personal knowledge base—things you've learned about yourself that should influence future decisions.

Add to this document during quarterly and annual reviews. Reference it before making major decisions.

---

## How to Use This Document

### Adding Insights
- After each quarterly or annual review, add 2-3 key insights
- Date each entry
- Be specific—vague insights are useless
- Include context about when/how you learned this

### Referencing Insights
- Before major decisions, scan relevant sections
- During weekly reviews, check if patterns are recurring
- When feeling lost, review your accumulated wisdom
